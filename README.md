<h2> Чтобы запустить проект, выполните следующие шаги:</h2>

1. Создайте контейнеры:

```make start```

2. Зайдите в контейнер

```make bash```

3. Установите библиотеки и создайте таблицы в базе данных, заполните их данными

``` composer install ``` <br>
```make migrate```<br>
```make seed```
